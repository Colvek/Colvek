## Principles

What I expect from myself, every commit.

| Principle | Standard | In practice |
|:---|:---:|:---|
| **TypeScript strict** | `strict: true` | Types are not documentation: they are contract |
| **Tests as specification** | ≥ 80 % coverage | Every bug is reproduced first with a failing test |
| **Atomic commits** | 1 : 1 | One commit, one purpose. Traceable from ticket to diff |
| **Measured performance** | LCP < 1.2 s | Lighthouse + Web Vitals on every PR. No blind optimisation |
